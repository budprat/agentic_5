## 🔧 Installation & Setup

### Prerequisites
- Python 3.9+
- Node.js 16+
- PostgreSQL 13+
- Redis 6+
- Docker (optional)

### Quick Start
```bash
# Navigate to project directory
cd linkedin-domination

# Install dependencies
pip install -r requirements.txt
npm install

# Set up environment variables
cp config/.env.example config/.env
# Edit config/.env with your API keys

# Initialize database
python phase-0/project_structure_init.py

# Run the system
python main.py
```

### Docker Setup
```bash
# Start all services
docker-compose up -d

# Verify services
docker-compose ps
```
